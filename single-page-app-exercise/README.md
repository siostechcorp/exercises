# Single Page Web-App Challenge


## Initialization

#### Development

- npm install bower grunt-cli -g

#### Data Serving

- npm install json-server -g

#### Project

- cd path/to/exercise/single-page-app-exercise/
- npm install
- bower install


## Viewing

- cd path/to/exercise/single-page-app-exercise/
- json-server -w data.json
- grunt serve


## Testing

- cd path/to/exercise/single-page-app-exercise/
- grunt test
